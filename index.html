Explain DB:

			Organized way to save data on a server. LIFEBLOOD of all business, to track employees, customers, products, services, supliers...

Explain MySQL:

			Number 2 among the must-used DB's in the entire world! Oracle is number 1 because it offers professional support,
			but is a paid service. MySQL is FREE! The inventor of MySQL named this after his first daughter, her name is "My."
			This same inventor later created MariaDB, after his second daughter "Maria."

Explain SQL:

			Structered Query Language. MySQL is the name of the database, while SQL is the language you use inside of that 
			database to answer questions -- ;ike "which products are slow sellers, which products are fast sellers.." SQL
			is an international standard language, works with MySQL, Oracle SQL, Microsoft SQL server, and many others.

Explain PHP:

			PHP is a programming language that connects front end to back end. PHP readsHTML forms, then sends that form 
			info to the server. PHP also reads the DB, ad sends that back to another HGML form. PHP is a server-side 
			language, totally secure and private - you can't see PHP when you do a "view source."

Explain MySQL Table:

			Has rows and columns, where data is organized in cells (rows are horizontal, columns are vertical). Looks
			like an Excel spreadsheet, but as you will learn it is actually very different from Excel.
